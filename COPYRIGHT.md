## Original Work

Markdraft began as a fork of [Grip](https://github.com/joeyespo/grip)
by Joe Esposito (Copyright (c) 2014-2022, MIT License). Grip's reader
abstractions and CLI design informed markdraft's early architecture.

The codebase has since been entirely rewritten: the Flask web framework
was replaced with Python's stdlib http.server, server-side markdown
rendering was replaced with client-side JavaScript, and all pip
dependencies were removed. No substantial original Grip code remains.
